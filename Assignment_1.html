b. Difference between <div> and <span>

<div> is a block-level tag used for large sections.

<span> is an inline tag used for small text styling.

<div>This is a section using div.</div>
<p>This is <span style="color:red;">highlighted text</span> using span.</p>

Q.2: CSS Concepts & Selectors

a. CSS Selectors 

p { color: blue; }        /* Element Selector */   Element selector applies to all tags 
.title { color: red; }    /* Class Selector */     class selector applies to multiple elements
#main { color: green; }   /* ID Selector */        ID selector applies to a unique element

b. Display Property

display: inline; → stays in the same line.

display: block; → starts on a new line and takes full width.

display: inline-block; → stays in line but allows width and height changes.

Q.4: Flexbox Layout

a.) what is flex box in css and why we used ? mention any two advantages?

Flexbox is a CSS layout method that helps arrange elements easily in rows or columns. It adjusts automatically to screen size and gives better control over alignment and spacing.

Advantages: (1) Makes responsive design easier. (2) Helps align and space items perfectly.
