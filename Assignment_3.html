Q (a) . Explain the difference between the following CSS selectors with examples: ?

  • Element selector===It is used to style all HTML elements of the same type, like all paragraphs or all headings. select by (tag).
  • Class selector=====It is used to style many elements that share the same class name. It allows you to apply the same style
    to multiple elements.select by (.).
  • ID selector=======It is used to style one unique element on a page. Each ID should be used only once.select by (#).

Q (b) .What is the difference between display: inline;, display: block; and display: inline-block;? ?

  • display: inline=====The element appears in the same line with others and does not start a new line.
  • display: block=====The element starts on a new line and takes the full width available.
  • display:inline-block==The element stays in line with others but allows you to set its width and height.
